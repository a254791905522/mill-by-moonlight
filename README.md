# Mill By Moonlight Release Package

App name: Mill By Moonlight

Bundle ID: com.rosewood.millbymoonlight.game

Device support: iPhone only

Gameplay fact card:
- Core action: Harness moon-driven tides to grind flour at the historic Eling tide mill.
- Main puzzle: Read the moon phase and engage the millstone at the right moment to maximize flour output.
- Tide model: Moon phase drives tide amplitude with half-day cycles and weather disturbance.
- Progression: 50 contracts across four millstone eras (Stone, Porcelain, Steel, Ceramic).
- Monetization and network: offline, no ads, no in-app purchases, no account, no analytics, no tracking.
